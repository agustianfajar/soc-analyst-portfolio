# SOC Analyst Portfolio

This repository is a personal SOC Analyst portfolio.

The purpose of this repository is to document hands-on labs, reports, and supporting materials related to Security Operations Center fundamentals.

Focus areas:
- Networking fundamentals
- Linux and Windows basics
- SIEM and log analysis
- Incident handling
- Threat intelligence

Repository structure:
- Labs: Hands-on lab exercises organized by topic
- Reports: Incident analysis and security reports
- Scripts: Supporting scripts for SOC-related tasks
- Resources: Study notes and reference materials
